## Install

### NPM

```bash
npm install @moontai0724/flickr-sdk
```

### Yarn

```bash
yarn add @moontai0724/flickr-sdk
```

### PNPM

```bash
pnpm add @moontai0724/flickr-sdk
```

## Usage

```typescript
import { add, divide, max } from "@moontai0724/flickr-sdk";

add(1, 2); // 3
divide(1, 2); // 0.5
max(1, 2); // 2
```
